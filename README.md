# E-Commerce Sales & Operations Analytics

This project analyzes the **Olist Brazilian E-Commerce dataset** to understand sales performance, delivery operations, and customer behavior.

The analysis was performed using **SQL for data exploration and business metrics**, and **Power BI for building an interactive dashboard**.

## Tools Used
- PostgreSQL
- SQL
- Power BI

## Key Insights
- Identified revenue trends and order patterns
- Analyzed delivery performance and delays
- Studied the impact of delivery time on customer reviews
- Found high-value customers contributing to revenue

## Dashboard Features
- Total Revenue and Orders KPIs
- Monthly Sales Trend
- Revenue by State
- Orders by Status
- Review Score vs Delivery Time
- Top Customers by Revenue

This project demonstrates an **end-to-end data analytics workflow from SQL analysis to Power BI visualization.**
